graph TD

1. Startup & Authentication

    Start((System Launch)) --> LoadData[FileHandler: Load Vehicles, Users, Transactions]
    LoadData --> LoginMenu{LOGIN MENU}

    LoginMenu -- 1. Register --> RegFlow[Enter Name, ID, Password] --> SuccessReg[Account Created] --> LoginMenu
    LoginMenu -- 2. Login --> Auth[Enter Credentials] --> RoleCheck{System Role?}
    LoginMenu -- 3. Exit --> SaveExit[FileHandler: Save All Data & Close]


2. Role Routing

    RoleCheck -- Admin --> AdminDash[ADMIN CONTROL PANEL]
    RoleCheck -- Customer --> CustDash[CUSTOMER DASHBOARD]


3. Admin Journey

subgraph Admin_Workflows [Admin Actions]
    AdminDash --> AddV[1. Add Vehicle]
    AdminDash --> RemV[2. Remove Vehicle]
    AdminDash --> SPM[3. Sale / Purchase Module]
    AdminDash --> ViewR[4. View All Records]
    AdminDash --> LogoutA[5. Logout]
end

Admin Detail: Vehicle Categories

    AddV --> CatMenu{Select Category}
    CatMenu -- Sedan/Hatch/Coupe --> V1[ID, Model, Year, Rent/Day]
    CatMenu -- SUV/X-Over/Pickup --> V2[ID, Model, Year, Rent/Day, Seats]
    CatMenu -- Van/Minivan/Bus --> V3[ID, Model, Year, Rent/Day, Pax Capacity]
    CatMenu -- Luxury/Convertible --> V4[ID, Model, Year, Rent/Day, Features]
    V1 & V2 & V3 & V4 --> SaveV[Add to Fleet & Save] --> AdminDash

    RemV --> EntID_Del[Enter Vehicle ID] --> DelConf[Delete from Fleet] --> AdminDash

Admin Detail: Sale/Purchase Module

    SPM --> SPM_Sub{Action?}
    SPM_Sub -- Sell Vehicle --> SellV[ID + Buyer ID + Price] --> ReceiptS[Gen Sale Receipt & Mark      'Sold'] --> AdminDash
    SPM_Sub -- Purchase Vehicle --> PurV[Supplier + Details + Price] --> ReceiptP[Gen Purchase Receipt & Add to Fleet] --> AdminDash

    ViewR --> DispAll[Show All Vehicles: Avail/Rented/Sold & All Users] --> AdminDash
    LogoutA --> LoginMenu


4. Customer Journey

subgraph Customer_Workflows [Customer Actions]
    CustDash --> Search[1. Search Vehicles]
    CustDash --> Rent[2. Rent a Vehicle]
    CustDash --> Return[3. Return a Vehicle]
    CustDash --> Trip[4. Plan a Trip]
    CustDash --> History[5. Rental History]
    CustDash --> LogoutC[6. Logout]
end

Customer Detail: Search Filters

    Search --> FilterMenu{Pick Filter}
    FilterMenu -- Model --> F1[Enter Model Name]
    FilterMenu -- Price --> F2[Enter Min/Max Range]
    FilterMenu -- Category --> F3[Select Category Type]
    FilterMenu -- Availability --> F4[Filter 'Available' Only]
    FilterMenu -- Year --> F5[Enter Year Range]
    F1 & F2 & F3 & F4 & F5 --> ShowRes[Display Matches] --> SearchAgain{Search Again?}
    SearchAgain -- Yes --> Search
    SearchAgain -- No/Back --> CustDash

Customer Detail: Rent & Return
    Rent --> EntRentID[Enter Vehicle ID] --> RentCheck{Is Available?}
    RentCheck -- Yes --> RTxn[Create Rental Transaction] --> MarkRent[Mark 'Rented'] --> CustDash   65     RentCheck -- No --> RentErr[Show Error Message] --> CustDash

    Return --> EntRetID[Enter Vehicle ID] --> InspSheet[OPEN INSPECTION SHEET]
    InspSheet --> Fields[Enter: Date, Fuel, Mileage, Damage] --> Eval[Enter: Remarks : Good/Fair/Poor]
    Eval --> Calc[Enter Total Days Used] --> Bill[Display Itemized Bill: Days x Rate + Fees]
    Bill --> MarkAvail[Mark 'Available'] --> CustDash

Customer Detail: Trip & History
    Trip --> TripFields[Start, Dest, Dist, Budget, Pax] --> Recommend[Filter Fleet for Matches] --> DispRec[Show Recommendations] --> CustDash
    History --> DispPast[List Past Rentals, Receipts, Period, Bills] --> CustDash

    LogoutC --> LoginMenu
